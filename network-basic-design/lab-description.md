# Network Basic Design Lab

## Overview
This lab was created as part of my CCNA hands-on practice using Cisco Packet Tracer.
The goal of this lab is to simulate a small campus network and practice both Layer 2 and Layer 3 concepts together.

## Technologies Used
* VLAN
* Trunking
* STP (Spanning Tree Protocol)
* EtherChannel
* Inter-VLAN Routing
* Static Routing
* Router-on-a-Stick (ROAS)

## Lab Objectives
* Segment network using VLANs
* Configure trunk links between switches
* Prevent loops using STP
* Aggregate links using EtherChannel
* Enable communication between VLANs
* Configure routing between networks

## Tools
* Cisco Packet Tracer

## Notes
This lab helped me understand how multiple networking concepts work together in a real-world scenario rather than individually.
