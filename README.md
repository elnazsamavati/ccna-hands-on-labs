# CCNA Hands-On Labs

## About
This repository contains my hands-on networking labs as I progress through CCNA topics. I use both Cisco Packet Tracer and EVE-NG to simulate real-world network scenarios.

---

## Progress
I update this repository weekly with new labs, gradually covering the complete CCNA curriculum through practical scenarios.
